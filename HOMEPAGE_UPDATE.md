
### 核心功能卡片

#### 1. 日志追踪查询 🔍
- 支持 REQ_SN 精准查询
- 支持商户号过滤
- 支持日志时间定位（YYYYMMDDHH 格式）
- 自动提取 TraceID
- 多 TraceID 分组展示（重复查询场景）
- 支持导出 CSV

#### 2. 配置管理 ⚙️
- **交易类型管理**
  - 新增/编辑/删除交易类型
  - 配置交易代码和名称
  - 关联多个应用

- **应用日志配置**
  - 新增/编辑/删除应用配置
  - 路径验证功能
  - 实时检测路径状态

- **系统配置**
  - 应用基础配置
  - 日志级别设置
  - 搜索参数配置

#### 3. 技术特性 🚀
- 多行日志解析（XML 报文）
- GBK/UTF-8 编码自动检测
- 大文件流式处理
- TraceID 链路追踪
- 重复查询识别（多 TraceID）
- 时间范围日志文件定位

### 快速开始指引
```
1. 输入查询条件 → 2. 自动提取 TraceID → 3. 查看完整链路
```

### 使用说明
- 日志时间格式：10 位数字 YYYYMMDDHH
- TraceID 提取：日志格式第 3 个参数
- 重复查询：同一个 REQ_SN 可能对应多个 TraceID
- 多行日志：以时间戳开头作为新日志的开始
