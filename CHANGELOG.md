# 更新日志

所有重要的项目变更都将记录在此文件中。

## [2.3.0] - 2026-04-09

### Added
- ✨ 交易类型追踪支持多 TraceID 分组展示（重复查询场景）
- ✨ 首页功能说明全面更新，详细介绍各功能模块
- ✨ Gzip 压缩日志文件支持（.log.gz）
- ✨ 标准项目文档（README.md、LICENSE、.gitignore）

### Changed
- 🔄 优化交易类型追踪 API，支持按 TraceID 分组返回
- 🔄 前端折叠面板展示多个 TraceID 分组
- 🔄 文件读取支持 .log 和 .log.gz 格式

### Fixed
- 🐛 修复 TraceAnalyzer 缺少 load_transaction_types 方法的问题
- 🐛 修复中文日志显示乱码问题（GBK 编码支持）
